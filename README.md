### Installation

_To work with the project you must install the following packages_

2. install django
   ```sh
    pip install django
   ```
   
3. Install restframework
   ```sh
   pip install rest framework
   ```
   
4. install pytest
   ```sh
   pip install pytest django-pyest
   ```
